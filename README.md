# StudyAngular9
測試和學習Angular9

先使用npx來建立測試一下angular9的特性

```
npx @angular/cli new TestUIDashboard
```

 安裝flex-layout模組

```
npm i -s @angular/flex-layout
```

在`SharedAngularMaterialModule`使用以下的

```
import { ScrollingModule } from '@angular/cdk/scrolling';
```
